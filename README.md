# Props.

| Props                     | Required | Default Value |
| ------------------------- | -------- | ------------- |
| **variant** number        | No       | default       |
| **disabled** number       | No       | false         |
| **disabledShadow** number | No       | false         |
| **color** number          | No       | default       |
| **size** number           | No       | sm            |

## Usage

```jsx
import Button from "dev-challenge-button"

function App() {
  return (
    <>
      <Button variant='text' disabled />
    </>
  )
}
```
